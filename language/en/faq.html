Q: What is LUFS?

Imagine LUFS(Loudness units relative to full scale) being a relative volume from the baseline(0 LUFS)，while 0 LUFS represent the loudest possible signal, so any other signal should be weaker than 0 LUFS, thus the volume of your audio is negative LUFS.

Q: What is the unit dB used in VolumeUp?

The dB used here isn't the one we measured in our daily life with unit dBA, here the dB is only for user to easily understand while determine target value. Since there is no direct transformation formula to convert LUFS to dBA, so we set 0 LUFS to be equal to 103dB in order to shift the value of volume to the range we familiar with.
Maybe custom shifting value will be a feature in the future.
