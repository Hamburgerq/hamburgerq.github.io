Functions

- Multiple audio files adjustment

Analyze volume of imported files in real time, batch adjust to target volume.

- Support common used file format

Support input & output formats: WAV, MP3, FLAC, AIFF, AAC, OGG, MP4, MOV, MKV

- Audio clipping prevention

Prevent audio from clipping while boosting volume, while clipping sets to the maximum non clipping value.

- Customization

Support dark theme, multiple languages, changable loudness unit(LUFS or dB), target volume memorization.
